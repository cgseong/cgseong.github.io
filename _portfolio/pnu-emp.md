## PNU-EMP

부산대학교 EMP(Employee Management Platform) 시스템입니다.

### 주요 기능

- TypeScript 기반 웹 애플리케이션
- Vercel을 통한 자동 배포
- 부산대 관련 업무 관리 기능

### 기술 스택

- **언어**: TypeScript
- **배포**: Vercel
- **프레임워크**: Next.js
